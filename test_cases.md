# Тест-кейсы для проекта RetroBoard

## 1. Управление досками (Boards)

### TC-001: Создание новой доски
| Параметр | Значение |
|---|---|
| **ID** | TC-001 |
| **Приоритет** | High |
| **Предусловия** | Пользователь на главной странице, боковая панель видна |

| Шаг | Действие | Ожидаемый результат |
|---|---|---|
| 1 | Нажать кнопку «Новая доска» в боковой панели | Открывается модальное окно «Новая доска» |
| 2 | Ввести название доски (например «Sprint #43») | Поле ввода принимает текст |
| 3 | Нажать кнопку «Создать» или Enter | Доска создана, отображается в боковой панели, основная область показывает колонки по умолчанию |

### TC-002: Выбор существующей доски
| Параметр | Значение |
|---|---|
| **ID** | TC-002 |
| **Приоритет** | High |

| Шаг | Действие | Ожидаемый результат |
|---|---|---|
| 1 | В боковой панели нажать на название доски | Доска выбирается, отображается в основной области, имя появляется в топбаре |

### TC-003: Переименование доски через топбар
| Параметр | Значение |
|---|---|
| **ID** | TC-003 |
| **Приоритет** | Medium |

| Шаг | Действие | Ожидаемый результат |
|---|---|---|
| 1 | Выбрать доску | В топбаре отображается имя доски с полем ввода |
| 2 | Изменить текст в поле имени | Текст изменяется |
| 3 | Потерять фокус (blur) или нажать Enter | Имя доски обновлено, сохранено в хранилище |

### TC-004: Копирование доски
| Параметр | Значение |
|---|---|
| **ID** | TC-004 |
| **Приоритет** | Medium |

| Шаг | Действие | Ожидаемый результат |
|---|---|---|
| 1 | Выбрать доску | В боковой панели видна кнопка «Копировать доску» |
| 2 | Нажать кнопку «Копировать доску» | Открывается модальное окно с предзаполненным названием (оригинал + "(копия)") |
| 3 | Изменить название при необходимости | Поле редактируется |
| 4 | Нажать «Копировать» | Создается копия доски, все голоса карточек сброшены, новая доска отображается в боковой панели |

### TC-005: Удаление доски
| Параметр | Значение |
|---|---|
| **ID** | TC-005 |
| **Приоритет** | High |

| Шаг | Действие | Ожидаемый результат |
|---|---|---|
| 1 | Выбрать доску | В боковой панели у доски есть кнопка удаления (иконка крестика) |
| 2 | Нажать кнопку удаления | Открывается модальное окно подтверждения «Удалить "[название]"?» |
| 3 | Нажать «Отмена» | Окно закрывается, доска не удалена |
| 4 | Нажать «Удалить» | Доска удалена из хранилища и боковой панели; если это была последняя доска — показывается empty state |

### TC-006: Empty state (нет активных досок)
| Параметр | Значение |
|---|---|
| **ID** | TC-006 |
| **Приоритет** | Low |

| Шаг | Действие | Ожидаемый результат |
|---|---|---|
| 1 | Удалить все доски или не создавать новые | В основной области отображается empty state с иконкой, заголовком «Нет активной доски» и подсказкой |
